# HTML Starter

A simple starter project for practicing HTML and basic web page structure.

This repository is designed for learning and quick prototyping.  
It includes a clean HTML structure that can be used as a base for small websites, landing pages, or layout experiments.

## Features
- Semantic HTML structure
- Clean and readable markup
- Easy to extend with CSS and JavaScript
- Suitable for beginners and small projects

## Technologies
- HTML5

## Purpose
This project was created for learning and practicing front-end fundamentals, including page structure, elements, and semantic markup.

It can also be used as a starting point for simple static websites.
